Linear gradient

    "#000091", "#E1000F",
    output=ColorisatorFormat.HEX
    )

Linear Gradient with stops

gradient = Colorisator.gradient_stops(
    ["#000091", "#FFF", (225, 0, 15)],
    10,
    output=ColorisatorFormat.HEX
    )
